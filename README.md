# python---webscraper
# **Money Exchange Rates Scraping using Beautiful Soup and Python**

This project is a demonstration of scraping exchange rates information from websites using the Beautiful Soup library in Python. The data is scraped from a website and stored in for further processing and analysis. The project is implemented in Jupyter Notebook format.

## **Prerequisites**

- Python 3
- Jupyter Notebook
- Beautiful Soup
- Requests

## **Installation**

To install the required libraries, use the following pip command:

```
Copy code
pip install beautifulsoup4 requests

```

## **Usage**

1. Open the Jupyter Notebook file **`money_exchange_rates_scraping.ipynb`**
2. Run the cells to see the implementation of the project

## **Project Description**

The project consists of the following steps:

1. Importing required libraries
2. Defining a function to scrape exchange rate information from the website
3. Sending a request to the website and parsing the HTML content using Beautiful Soup
4. Extracting the relevant information from the parsed HTML content and storing it .
5. Cleaning the data and converting it according to the user needs.
